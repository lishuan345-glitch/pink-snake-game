# 粉红贪吃蛇游戏 🐍💕

一个可爱的粉红色风格贪吃蛇网页小游戏，带有动听的电子音乐背景。

## 游戏特色

### 🎨 视觉设计
- **粉红色可爱风格**: 全粉色主题，从深粉到浅粉的渐变设计
- **圆角设计**: 所有元素都采用圆角设计，更加可爱
- **光晕效果**: 食物带有粉色光晕特效
- **流畅动画**: 浮动、脉搏、缩放等多种CSS动画效果
- **可爱的蛇**: 蛇有眼睛，会根据移动方向转动

### 🎵 音效系统
- **背景音乐**: 使用Web Audio API生成的可爱电子旋律
- **音效**: 吃食物和游戏结束都有独特的音效
- **音乐控制**: 可随时开关背景音乐

### 🎮 游戏功能
- **经典玩法**: 传统贪吃蛇游戏机制
- **键盘控制**: 方向键控制蛇的移动
- **空格暂停**: 按空格键快速暂停/继续
- **分数系统**: 实时显示当前分数和最高分
- **本地存储**: 最高分自动保存
- **难度递增**: 分数越高，蛇移动越快
- **游戏结束界面**: 精美的游戏结束提示

### 📱 响应式设计
- 支持不同屏幕尺寸
- 移动端友好的布局

## 游戏规则

1. 使用方向键（↑ ↓ ← →）控制蛇的移动方向
2. 吃到食物（粉色圆点）会增加分数和蛇的长度
3. 撞到墙壁或自己的身体会导致游戏结束
4. 每50分蛇的速度会增加一次
5. 每吃一个食物获得10分

## 游戏控制

### 键盘控制
- `↑` - 向上移动
- `↓` - 向下移动
- `←` - 向左移动
- `→` - 向右移动
- `空格` - 暂停/继续游戏

### 按钮控制
- **开始游戏 🎮**: (欢迎界面) 进入游戏并开始
- **暂停**: 暂停/继续当前游戏
- **重新开始**: 重置游戏到初始状态
- **返回菜单**: 返回欢迎界面
- **再来一次**: (游戏结束界面) 重新开始游戏
- **🎵**: 开关背景音乐

## 如何运行

1. 直接在浏览器中打开 `index.html` 文件
2. 阅读欢迎界面的游戏说明
3. 点击"开始游戏 🎮"按钮进入游戏
